### O que é o projeto?
Ele é um To do List feito no modelo API para usuários que desejam criar uma rotina mais organizada.

### Git clone

Clone o projeto do Git e acesse a pasta do projeto.
```
git clone git@github.com:Willian9857/to_do.git
cd to_do
```
